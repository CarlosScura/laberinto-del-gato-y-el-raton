# laberinto-del-gato-y-el-raton
Repositorio para el Challenge 1.

## Juego del  gato y del ratón.

Se creó un juego donde un gato (usando el algoritmo minimax) persigue a un ratón (el jugador), dentro de un tablero de 5x5.

El gato tiene 6 turnos para atrapar al ratón, y el ratón los tiene para que el gato no lo atrape, si el ratón logra que el gato no lo atrape, gana la partida el ratón, si el gato lo atrapa, el ratón pierde.

## Funciona bien.

La implementación de minimax con profundidad limitada permitió que el gato anticipe movimientos y reduzca la distancia estratégicamente. También funcionó correctamente el movimiento del ratón.
